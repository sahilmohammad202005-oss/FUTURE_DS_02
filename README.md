# Telco Customer Churn Analysis Dashboard

## Overview
This project analyzes customer churn patterns in a telecom company using Power BI. The dashboard helps identify factors influencing customer retention and churn through interactive visualizations and key performance indicators (KPIs).

## Objectives
- Analyze customer churn behavior.
- Identify high-risk customer segments.
- Understand the impact of contract types, internet services, and payment methods on churn.
- Provide insights that can help improve customer retention.

## Tools Used
- Power BI
- Excel Web

## Key Metrics
- Total Customers
- Churned Customers
- Active Customers
- Churn Rate (%)

## Dashboard Features
- Interactive slicers for:
  - Gender
  - Contract Type
  - Senior Citizen Status
  - Internet Service

- Visualizations:
  - Customer Distribution by Contract Type and Churn
  - Customer Distribution by Payment Method and Churn
  - Churn Analysis by Monthly Charges
  - Internet Service Analysis
  - Customer Tenure Distribution

## Key Insights
- Month-to-month contract customers show higher churn compared to long-term contracts.
- Customers using fiber optic internet services have a higher churn tendency.
- Long-term contracts are associated with better customer retention.
- Churn patterns vary across payment methods and monthly charges.

## Project Outcome
The dashboard provides a clear overview of customer churn trends and helps businesses make data-driven decisions to improve customer retention strategies.
